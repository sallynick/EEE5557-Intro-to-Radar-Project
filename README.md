# EEE5557-Intro-to-Radar-Project
Radar trade study, including the final presentation, code, and supporting documents
The goal was to create a radar that was both compliant for certian parameters while also achieving goals
A MATLAB script was necessary to compute the design parameters and their effects

In hindsight, I would have kept the struct format off all my inputs and outputs, but I would have broken each function into its own .m file rather than using local functions.
The troubleshooting would have been easier, and I could have more easily modified the functions for other scripts that I used.
